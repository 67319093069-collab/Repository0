วิเคราะห์กระบวนการ

Input, Process และ Output ตามเอกสารออกแบบ

Input

  • ข้อมูลผู้ขาย
  • รายการสินค้า
  • จำนวนที่ต้องการ

Process

  • Login และตรวจสอบสิทธิ์
  • เลือกสินค้าและตรวจสต็อก
  • สร้างใบสั่งซื้อและอนุมัติ
  • บันทึกข้อมูลและออกรายงาน

Output

  • ใบสั่งซื้อ
  • รายงาน
  • ข้อมูลสต็อก
